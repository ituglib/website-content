Instructions for use of curl versions 7.70.0 and above

Use the OSS command
  tar xzf curl-package.tar.gz -C /usr/local -p --no-same-user
to install curl. Make sure to have the directory containing the OpenSSL 1.0.2 version of libcrypt.so in your _RLD_LIB_PATH variable when curl is used.

Due to new features added in the latest OSS System Libraries, binaries built with those will not function correctly on older RVUs. The following table lists the minimum RVU for each platform that can correctly run curl version 7.70.1, and above:

Platform Minimum RVU
TNS/E J06.22
TNS/X L19.08
TNS/V L19.08
